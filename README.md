# 📊 大学生个人情绪量化预警系统 (Emotion-Tracker)

> 基于多巴胺预测误差（RPE）与本地冷热数据分离架构的轻量级心理干预程序。

## 💡 项目背景与核心愿景
针对当代大学生面临的多维压力，本项目立足于**绝对数据主权**理念，旨在提供一个纯本地运行、高隐私安全的情绪趋势量化工具。

## 🚀 核心架构与技术创新 (Core Features)

*   **M1 模块 | 数据容错与插值降噪：** 采用线性插值算法自动拟合缺失值，保证情绪趋势曲线的连续性。
*   **M2 模块 | 动态基准线与 RPE 修正：** 引入**多巴胺奖赏预测误差**逻辑，精准放大情绪落差感，实现敏锐预警。
*   **M4 模块 | 极客级存储运维：** 面对 10MB 存储限制，引入 **LRU 淘汰策略** 与 **冷热数据分级架构**。

## 🛠️ 部署与运行指南 (Quick Start)
1. 下载并安装最新版的**微信开发者工具**。
2. 导入项目目录，并在设置中选择**测试号**运行。
3. 点击“编译”即可在模拟器中体验完整业务流。
